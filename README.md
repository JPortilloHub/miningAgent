## Mining Agent

### Goal
The objective behind this project is mining a 100% agentic workflow to demonstrate how process mining technology brings governance to Agentic AI. This workflow consists of an user input their credentials and motive to see if its healthcare insurance covers them.

### Notebooks
Insurance_Verification_Agent.ipynb contains the code which create the agent, its tools and the interaction with the users

Process_Mining.ipynb contains the code which process the agent logs and draw the workflow the agent followed with its respective variants

### Files

docs folder contains all the files the two previous notebooks need as inputs such as the insurance policy, prompts from patients or agent logs

Patient.csv and Patient.xlsx contain the list of 50 different policyholders

Medical Policy.docx contains the policy which the agent will be based on to make decisions

Prompts.csv contains a list of 115 prompts generated by different patients

agent_logs.csv contains the steps the agent has taken for each of those 115 prompts

test.py contains different casuistics to test the agent. It also incluse edge cases to evaluate how it reasons
